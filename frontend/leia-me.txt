*com o powershell administrador
  - navegar até a pasta anterior ao backend
  - digitar "npx create-react-app frontend" 
    - comando cria a pasta frontend preparada para o react 
  - acessar pasta frontend
  - digitar "code ."
    - comando abre pasta no VSCode

* VSCode abrir terminal
  - digitar "npm start"
    - comando inicia frontend

* Pacote de ícones
  - digitar "npm install react-icons"
    comando instala pacote de ícones    

* Rotas
  - digitar "npm install react-router-dom"
    comando instala pacote de navegação    

* Cliente HTTP
  - digitar "npm install axios"
    comando install os componentes do cliente HTTP
